Problem Statement

In modern recruitment, both candidates and recruiters face major inefficiencies:

  • Candidates struggle to create ATS-friendly resumes
  • They do not know what skills are missing
  • They apply to wrong jobs
  • Recruiters manually screen hundreds of resumes

This leads to time wastage, poor hiring quality, and high rejection rates.

Proposed Solution

We developed an end-to-end AI career assistant platform that builds resumes using an AI chatbot, analyzes resumes using NLP, calculates ATS compatibility score, recommends suitable jobs, and suggests improvements to increase selection chances.

The system works as both a candidate tool and a recruiter screening tool.

System Overview

Candidate Workflow

User -> AI Chatbot -> Resume Generation -> ATS Score -> Skill Gap -> Job Recommendation

Recruiter Workflow

Upload Job Description -> Upload Resumes -> Auto Ranking -> Best Candidate Match

Core Features

1. AI Resume Builder (Chatbot Based)

Conversational question-answer system that collects candidate information step-by-step and generates resumes from templates with PDF, DOCX, and TXT export.

2. Resume Analyzer (NLP Based)

Resume parsing with spaCy, text preprocessing, skill and experience extraction, and category classification using ML models.

3. ATS Score Calculator

Compares resume with job description using TF-IDF and cosine similarity, then highlights missing keywords.

4. Job Recommendation System

Content-based recommendations match resume skills with job data, rank top opportunities, and suggest skill improvements.

5. Recruiter Auto-Screening Dashboard

Upload multiple resumes, automatically rank candidates, and find the best candidate instantly.
